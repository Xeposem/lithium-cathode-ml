## Cross-Phase Integration Findings

### Critical

1. **Pipeline config loading broken** -- `pipeline.py` extracts config sections from `data.yaml` but they live in separate YAML files
2. **Pipeline data handoff broken** -- reads `DataCache("cleaned_records")` but fetch saves to `data/processed/materials.json`
3. **Baseline results path mismatch** -- saved to `data/results/baseline_results.json`, loaded from `data/results/baselines/baseline_results.json`
4. **Dashboard pages render blank** -- `predict.py` and `crystal_viewer.py` use `if __name__ == "__main__"` guard

### Major

5. **Wrong import in model_loader** -- imports `structure_to_pyg_data` (nonexistent), should be `structure_to_graph`
6. **MEGNet checkpoint format mismatch** -- saves raw state_dict, loader expects `{"model_state_dict": ...}` wrapper

### Moderate

7. **Dashboard data path mismatch** -- `get_cached_records()` reads from cache path, fetch saves to processed path
