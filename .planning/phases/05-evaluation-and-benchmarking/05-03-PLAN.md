  <action>
    Create `cathode_ml/pipeline.py`:

    1. Imports: argparse, logging, pathlib.Path (at top). ALL heavy imports (data, models, evaluation) are LAZY inside stage functions to avoid startup overhead (locked decision pattern from prior phases).

    2. `build_parser() -> argparse.ArgumentParser`:
       - prog="cathode-ml-pipeline"
       - description="Run the full cathode ML pipeline end-to-end"
       - --models: nargs="+", choices=["rf","xgb","cgcnn","megnet"], default=all four
       - --skip-fetch: store_true, help="Skip data fetching (use cached data)"
       - --skip-train: store_true, help="Skip training (use saved checkpoints/results)"
       - --seed: int, default=42
       - --config-dir: default="configs"

    3. Stage functions (each with lazy imports inside):
       - `run_fetch_stage(args)`: Import and call cathode_ml.data.fetch.main()
       - `run_featurize_stage(args)`: Log that featurization happens during training (not a separate stage in current architecture -- data is featurized inline by each training orchestrator)
       - `run_train_stage(args)`: Load configs, load records from cache. For each model in args.models: call the appropriate training function (run_baselines, train_cgcnn, train_megnet). Use lazy imports for each model type. Skip models not in args.models.
       - `run_evaluate_stage(args)`: Import evaluation module, call generate_all_tables(). Import plots, call apply_nature_style(), plot_bar_comparison(), plot_learning_curves(). Log completion.

    4. `run_pipeline(args)`:
       - Define stages list: [("Fetching Data", run_fetch_stage, not args.skip_fetch), ("Featurizing", run_featurize_stage, True), ("Training Models", run_train_stage, not args.skip_train), ("Evaluating", run_evaluate_stage, True)]
       - Loop with stage banners: logger.info("=== Stage %d/%d: %s ===", i, total, name)
       - For skipped stages: logger.info("=== Stage %d/%d: %s (SKIPPED) ===", ...)

    5. `main()`:
       - logging.basicConfig(level=logging.INFO, format="%(levelname)s - %(message)s")
       - parser = build_parser()
       - args = parser.parse_args()
       - run_pipeline(args)

    Create `cathode_ml/__main__.py`:
       - Simple: `from cathode_ml.pipeline import main; main()`
       - This enables `python -m cathode_ml` as the top-level entry point

       - Verify skip logic: when skip_fetch=True, fetch stage not called
       - Verify stage banner logging: capture log output, check for "=== Stage" pattern
       - Do NOT actually run training/fetching in tests -- mock everything
