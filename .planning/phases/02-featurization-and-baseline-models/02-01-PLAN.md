Create `configs/features.yaml` with sections: graph (cutoff_radius: 8.0, max_neighbors: 12, gaussian: {dmin: 0.0, dmax: 5.0, num_gaussians: 80}, node_feature_dim: 100), composition (preset: "magpie", nan_strategy: "median"), splitting (train_size: 0.8, val_size: 0.1, test_size: 0.1, group_key: "reduced_formula"), target_properties list (formation_energy_per_atom, voltage, capacity, energy_above_hull).

    3. Create `cathode_ml/features/graph.py` with three functions:
       - `gaussian_expansion(distances: Tensor, dmin, dmax, num_gaussians) -> Tensor`: Expand distances into Gaussian basis. centers = linspace(dmin, dmax, num_gaussians), gamma = 1/(centers[1]-centers[0]), return exp(-gamma * (distances.unsqueeze(-1) - centers)**2).
       - `structure_to_graph(structure: Structure, config: dict) -> Data`: Convert pymatgen Structure to PyG Data. Read cutoff_radius, max_neighbors, Gaussian params from config["graph"]. Node features: one-hot atomic number (dim=100). Edges: use structure.get_all_neighbors(cutoff, include_index=True), sort by distance, take max_neighbors per atom. Edge features: gaussian_expansion on distances. Return Data(x=x, edge_index=edge_index, edge_attr=edge_attr).
       - `validate_graph(data: Data) -> tuple[bool, str]`: Check x exists and non-empty, edge_index exists and non-empty, all nodes appear in edge_index (no isolated nodes), edge_attr count matches edge count.

    4. Update `tests/conftest.py`: Add a `sample_pymatgen_structure` fixture that converts `sample_structure_dict` to a pymatgen Structure via `Structure.from_dict()`. Add a `features_config` fixture that returns the parsed features.yaml dict.

    5. Create `tests/test_graph.py` with tests per the behavior spec above.

    6. Update `requirements.txt`: Add torch>=2.2.0, torch-geometric>=2.7.0 (do NOT add torch-scatter/torch-sparse/etc -- those are install-time companions handled by pip, not requirements.txt entries).
