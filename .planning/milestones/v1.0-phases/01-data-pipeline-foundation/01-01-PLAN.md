  <behavior>
    - DataCache(cache_dir) creates the directory if it does not exist
    - cache.cache_key("mp", {"elements": ["Li"]}) returns a deterministic string based on source + params hash
    - cache.has(key) returns False for missing key, True after save
    - cache.save(key, data, metadata) writes JSON with timestamp, metadata, and data fields
    - cache.load(key) returns the saved data dict (not the wrapper with timestamp)
    - cache.load(key) for nonexistent key raises FileNotFoundError
    - Saving the same key twice overwrites the previous data
  </behavior>
  <action>
    Create the DataCache class in cathode_ml/data/cache.py following the research pattern:

    - __init__(self, cache_dir: str): Store Path(cache_dir), call mkdir(parents=True, exist_ok=True)
    - cache_key(self, source: str, params: dict) -> str: JSON-serialize params with sort_keys=True, compute MD5 hex digest, return f"{source}_{digest}"
    - has(self, key: str) -> bool: Check if cache_dir/f"{key}.json" exists
    - save(self, key: str, data: dict, metadata: dict = None) -> None: Write JSON with {"timestamp": datetime.now().isoformat(), "metadata": metadata or {}, "data": data} to cache_dir/f"{key}.json"
    - load(self, key: str) -> dict: Read JSON from cache_dir/f"{key}.json", return the "data" field only (not the full wrapper)
    - clear(self, key: str) -> None: Delete cache file if exists

    Use json module with indent=2 for human-readable cache files. Import hashlib for MD5.

    Write tests FIRST in tests/test_cache.py, then implement cache.py.
