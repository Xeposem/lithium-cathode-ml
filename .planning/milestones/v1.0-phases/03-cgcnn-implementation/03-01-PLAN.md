         - `__init__(self, node_feature_dim, edge_feature_dim, hidden_dim=128, n_conv=3, n_fc=1, batch_norm=True)`
         - `self.embedding = nn.Linear(node_feature_dim, hidden_dim)`
         - `self.convs = nn.ModuleList([CGConv(channels=hidden_dim, dim=edge_feature_dim, batch_norm=batch_norm) for _ in range(n_conv)])`

         - Reads hidden_dim, n_conv, n_fc, batch_norm from cgcnn_config["model"]
         - Returns CGCNNModel instance
