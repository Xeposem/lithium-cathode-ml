# Data Sources

This project uses battery material data from three sources. If you use
this data, please cite the original sources as described below.

## Materials Project

- **URL**: https://materialsproject.org
- **Access**: MP API (`mp-api` Python client)
- **Citation**: A. Jain et al., "Commentary: The Materials Project: A
  materials genome approach to accelerating materials innovation,"
  *APL Materials*, vol. 1, no. 1, p. 011002, 2013.
  DOI: [10.1063/1.4812323](https://doi.org/10.1063/1.4812323)

## Open Quantum Materials Database (OQMD)

- **URL**: https://oqmd.org
- **Access**: OQMD REST API
- **Citation**: S. Kirklin et al., "The Open Quantum Materials Database
  (OQMD): assessing the accuracy of DFT formation energies," *npj
  Computational Materials*, vol. 1, p. 15010, 2015.
  DOI: [10.1038/npjcompumats.2015.10](https://doi.org/10.1038/npjcompumats.2015.10)

## Battery Archive

- **URL**: https://www.batteryarchive.org
- **Data**: Cell-level cycling data (charge/discharge capacity, voltage,
  current) for LCO, NMC, NCA, and LFP cathode chemistries.
- **General citation**: Please cite www.batteryarchive.org as the source
  from where the data was downloaded.

Individual study citations are required per group guidelines:

### CALCE (Center for Advanced Life Cycle Engineering)

- **Chemistries**: LCO
- **Guidelines**: https://web.calce.umd.edu/batteries/data.htm

### HNEI (Hawaii Natural Energy Institute)

- **Chemistries**: NMC/LCO blend
- **Citation**: B. Bole, C. Kulkarni, and M. Daigle, "Intrinsic
  Variability in the Degradation of a Batch of Commercial 18650
  Lithium-Ion Cells," *Energies*, vol. 11, no. 5, p. 1031, 2018.
  https://www.mdpi.com/1996-1073/11/5/1031

### Oxford University

- **Chemistries**: LCO
- **Citation**: Oxford Battery Degradation Dataset 1.
- **Repository**: https://ora.ox.ac.uk/objects/uuid:03ba4b01-cfed-46d3-9b1a-7d4a7bdf6fac

### SNL (Sandia National Laboratories)

- **Chemistries**: LFP, NCA, NMC
- **Citation**: J. Lamb et al., "Degradation of Commercial Lithium-ion
  Cells as a Function of Chemistry and Cycling Conditions,"
  *Journal of The Electrochemical Society*.
  https://iopscience.iop.org
- **Data repository**: https://data.mendeley.com/datasets/sn2kv34r4h/

### University of Michigan
